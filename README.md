# 🌦 Real-Time Weather Forecasting System

## 📌 Project Overview
The **Real-Time Weather Forecasting System** is a Java desktop application developed using **Java Swing**, **OpenWeather API**, and **Gson**. It provides live weather information for any city entered by the user through a simple and interactive graphical user interface.

---
## 🚀 Features

- 🌍 Search weather by city name
- 🌡 Display current temperature
- 💧 Display humidity
- 🌬 Display wind speed
- ☁ Display weather condition
- 🖥 User-friendly Java Swing GUI
- 🔄 Real-time weather data using OpenWeather API

---

## 🛠 Technologies Used

- Java
- Java Swing
- OpenWeather API
- Gson Library
- HTTP URL Connection
- JSON Parsing

## ▶️ How to Run

1. Clone this repository.
2. Open the project in VS Code or Eclipse.
3. Add the Gson library to the project.
4. Replace the OpenWeather API key with your own.
5. Run `Main.java`.
